### 1\. Clone this Repository

```
git clone https://github.com/fgvlasov/eco-prog-web
```


### 2\. Install dependencies

```
npm install
```

### 3\. Run Dev Server

```
npm run dev
```

### 4\. Make own branch for work from dev branch, do not use main (it is for deploying)


```
git switch dev
git branch dev_Val_texts
```
